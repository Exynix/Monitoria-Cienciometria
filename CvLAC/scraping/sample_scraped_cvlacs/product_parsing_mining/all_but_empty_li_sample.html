    High Performance Computing algorithms for the RefficientLib software
    Inicio: Octubre  2016 Fin: Marzo  2017 Duración 
    Resumen

    This project aims to directly compute the acoustic pressure waves solving the compressible Navier-Stokes equations written in conservative variables. The finite element formulation used for the spatial approximation is based on the Variational Multi-Scale (VMS) framework. In order to apply the numerical compressible flow model to simulate the flow and the acoustic waves inside the vocal tract, some challenges arise. Because the flow features must be accounted for up to acoustic frequencies and turbulent scales, the problem lead to the need of fine meshes and thus a modern supercomputing facility. The parallel finite element compressible flow code developed for the project has already been tested in Marenostrum supercomputer. The code scalability has been tested up to 2048 cpus. Very good weak scalability (almost flat cpu time vs number of processors plots for constant per processor workload) has been obtained. Numerical developments such as parallel mesh adaptivity techniques and refinement algorithms are also needed because turbulent regions of the flow are not known a priori and may be resolved by coarse meshes. A parallel mesh adaptivity method based on the VMS framework has already been implemented and tested in up to 64 processors with good scalability results. The refinement procedure is designed to be dynamically run through the simulation and to preserve the work balance between processors. The objective is to improve the compressible model accuracy, specifically, capturing the acoustic frequencies in the sound production inside the vocal tract with the developed local mesh refinement techniques. The method is expected to produce better results in comparison with the static mesh solutions together with computational savings.

    Resumen

    La crisis medioambiental y la futura escasez de las fuentes de energía fósil ha llevado a que se popularice el desarrollo de tecnologías para el aprovechamiento de fuentes renovables, tales como las turbinas eólicas.
